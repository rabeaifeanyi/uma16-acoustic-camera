# A Low-Cost Acoustic Camera

## Requirements

See also ba_env.yaml.

- acoular
- acoupipe
- modelsdfg (dev)
- opencv  (must be compatible with tensorflow 2.15, e.g. version 4.9)
- tensorflow (version 2.15)

## Required Hardware

- UMA-16 microphone array
- USB-Camera

## Run application

`bokeh serve --show app.py`
